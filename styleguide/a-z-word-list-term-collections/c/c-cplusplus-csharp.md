# C, C++, C\#

It's OK to use *C, C++,* and *C\#* as adjectives. Don't form hyphenated modifiers with them, such as *C-based* or *C-callable*.

**Examples**  
a C program  
a program based on C++   
built with exported functions that are callable by C
